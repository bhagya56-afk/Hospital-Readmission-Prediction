# Hospital Readmission Prediction

## Project Overview
This project predicts whether a diabetic patient will be readmitted to the hospital using Machine Learning.

## Dataset
Diabetes Hospital Readmission Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib

## Workflow
1. Data Preprocessing
2. Feature Engineering
3. Train-Test Split
4. Feature Scaling
5. XGBoost Model Training
6. Performance Evaluation

## Results
The model predicts hospital readmissions based on patient medical records and demographic information.
